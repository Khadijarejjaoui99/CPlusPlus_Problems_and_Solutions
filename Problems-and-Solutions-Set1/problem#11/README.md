## Problem #11: Average Pass/Fail

Write a program to ask the user to enter 3, then print average of the entered
average, if the average >= 50 print "Pass" otherwise print "Fail" .
<br>Example Input:
<br>50
<br>60
<br>70
<br>Output:
<br>60
<br>"Pass"
